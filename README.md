## About app-ads.txt

The `app-ads.txt` file is used to declare authorized digital sellers for app inventory. This helps prevent ad fraud and ensures that only authorized parties can sell ad space in the Diarys Club app.

For more information about app-ads.txt, visit:
- [IAB Tech Lab app-ads.txt Specification](https://iabtechlab.com/ads-txt/app-ads-txt/)
- [Google AdMob app-ads.txt Guide](https://support.google.com/admob/answer/9363762)
